# 考场编排客户端

> An electron-vue sqlite3 project

#### Build Setup

``` bash
# 安装依赖
npm install

# 本地开发
npm run dev

# 本地打包（安装程序）
npm run build

# 本地打包（直接目录）
npm run build:dir

```
